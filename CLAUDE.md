# GymApp — Design Guidelines

Use the following skills for all UI work:
- emil-design-eng (motion, polish, invisible details)
- impeccable (anti-generic design patterns)
- redesign-existing-projects (audit before rewriting)

## Design Dials
DESIGN_VARIANCE: 5
MOTION_INTENSITY: 6
VISUAL_DENSITY: 6

## Stack
- Vite + React + TypeScript
- Use native CSS/Tailwind for styling, no extra UI libraries unless already present

## Goal
Premium gym admin feel — dark, structured, con energía.
Audit existing UI first, then add motion and polish.
Do not rewrite functionality, only improve visual quality and interactions.
